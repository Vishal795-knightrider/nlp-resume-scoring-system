// how it works

📄

Step 0 — PDF Upload (New!)

Upload a PDF resume. The server extracts all text using pypdf (page by page), then fills the resume field automatically. Manual text input still works as a fallback.

🧹

Step 1 — Preprocess Text

Both texts are lowercased, special characters are removed, and common stopwords (like "the", "is", "and") are filtered out, leaving only meaningful terms.

📊

Step 2 — TF-IDF Vectorization

Each word gets a weight. TF (how often it appears) × IDF (how rare it is globally). This highlights important keywords like "Python" or "Kubernetes" over generic words.

📐

Step 3 — Cosine Similarity

Each document becomes a vector. Cosine similarity measures the angle between them. Angle = 0° means perfect match (100%). It's length-independent, so short vs long docs are compared fairly.

Built with Python · sklearn · pypdf · Flask  |  Intelligent Matching
